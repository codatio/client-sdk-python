# BillStatus

Current state of the bill. If creating a bill the status must be `Open`.


## Values

| Name             | Value            |
| ---------------- | ---------------- |
| `UNKNOWN`        | Unknown          |
| `OPEN`           | Open             |
| `PARTIALLY_PAID` | PartiallyPaid    |
| `PAID`           | Paid             |
| `VOID`           | Void             |
| `DRAFT`          | Draft            |
