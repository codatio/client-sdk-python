# FeatureState

The current release state of the feature.


## Values

| Name              | Value             |
| ----------------- | ----------------- |
| `RELEASE`         | Release           |
| `ALPHA`           | Alpha             |
| `BETA`            | Beta              |
| `DEPRECATED`      | Deprecated        |
| `NOT_SUPPORTED`   | NotSupported      |
| `NOT_IMPLEMENTED` | NotImplemented    |
